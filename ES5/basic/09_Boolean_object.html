Boolean 오브젝트 - 빌트인 오브젝트 - true와 false 처리 - 값이 있으면 true로 인식 - false 인식 기준 - undefined, null, NaN - 빈 문자열, 숫자 타입의 0 - 숫자 값 변환 기준 - true를 1로, false를 0으로 변환 new Boolean() - Boolean 인스턴스 생성 - 파라미터 값을 true 또는 false로 변환하여 프리미티브에 설정 ex) var value = [undefined, null, NaN, 0, ""]; // false - 문자열이면서 값이 있으면 true로 변환 ex) var value = [12, "1", "0", "false"]; // true Boolean() - Boolean 값으로 변환 - 인스턴스를 생성하지 않고 true 또는 false로 변환 toString() - data 위치의 true, false를 문자열로 변환 - 즉 "true", "false"로 변환 valueOf() - Boolean 인스턴스의 프리미티브 값 반환 - 즉 true 또는 false 반환
