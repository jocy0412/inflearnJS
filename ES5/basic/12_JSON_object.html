JSON 오브젝트 JSON 오브젝트 개요 - JavaScript Object Notation - 빌트인 오브젝트 - new 연산자로 인스턴스 생성 불가 - JSON 주요 기능 - 데이터 송수신 변환 기준 - 텍스트이므로 전송 속도가 빠름 - 파일 확장자 : json, txt도 사용 가능 - JS 데이터 타입 지원 - 다른 언어도 사용하지만, 완전하게 같지 않을 수도 있음 stringify() - JS 타입을 JSON 타입의 문자열로 변환 JSON.stringify() 형태로 호출 - 모든 문자열은 ""을 사용하여 반환 - Infinity, NaN, null은 null로 변환 - undefined 변환 - 값 하나이면 그대로 변환 [코드] console.log(JSON.stringify(undefined)); // undefined - 배열 안에 있으면 null로 변환 [코드] console.log(JSON.stringify([undefined])); // [null] - 프로퍼티 값이면 프로퍼티 제외시킴 프로퍼티 이름도 없어지므로 주의 [코드] console.log(JSON.stringify({value: undefined})); // {} - true, false는 변환하지 않음 첫번째 파라미터(변환 대상) 두번째 파라미터(함수 또는 배열 opt) [코드] 두번째 파라미터에 함수 작성 var data = {book: '책', point: 55}; function replace(key, value){ // point 값 55를 11로 변경 return key === "point" ? 11 : value; }; var result = JSON.stringify(data, replace); console.log(result); // {"book":"책","point":11} [코드] 두번째 파라미터에 배열 작성 (배열 프로퍼티 이름과 같은것만 출력) var data = {book: '책', point: 11, amount: 90}; var result = JSON.stringify(data, ['book', 'amount']); console.log(result); // {"book":"책","amount":90} 세번째 파라미터(가독성을 위한 구분자 opt) [코드] 세번째 파라미터에 줄 분리 작성 var data = {sports: 'soccer', time: 90}; var result = JSON.stringify(data, "", '\n'); console.log(result); // { "sport":"soccer", "time":90 } [코드] 세번째 파라미터에 들여쓰기 숫자 var data = {sports: 'soccer', time: 90}; var result = JSON.stringify(data, "", 4); console.log(result); // { "sport":"soccer", "time":90 } [코드] 세번째 파라미터에 삽입할 문자 작성 var data = {sports: 'soccer', time: 90}; var result = JSON.stringify(data, "", 4); console.log(result); // { ##"sport":"soccer", ##"time":90 } JSON 타입 파싱 parse() - JSON 타입을 JS 타입으로 변환 - 값이 String타입의 숫자라도 타입이 숫자로 변환 - 값이 true/false면 Boolean 타입으로 변환 - 값이 배열이면 배열, 이 때 배열에 작성된 String 타입의 숫자는 숫자로 변환되지 않음 - 값이 Object일때 JS는 프로퍼티 이름에 큰 따옴표를 사용하지 않으므로 큰 따옴표가 표시되지 않음 [코드] 값이 숫자일 때 var value = "123"; try { var result = JSON.parse(value); } catch(e) { console.log("JSON 파싱 에러"); } console.log(result); // 123 console.log(typeof result); // number - 작성 주의 - "123."을 "123.0"으로 작성 - "0123"처럼 첫 자리에 0 사용 불가 - 대문자 "NULL" 사용불가, "null" 사용 - 10진수 사용 - 두번째 파라미터 작성 - 첫번째 파라미터의 JSON 문자열을 파싱하면 {book: "책", movie: "영화"}형태가 됩니다. - 파싱한 오브젝트를 하나씩 읽어가면서 두번째 파라미터의 함수를 실행 - 함수에서 값을 반환하면 파싱 결과인 result 변수에 반영 - 값을 반환하지 않거나 undefined를 반환하면 프로퍼티가 제외되므로 반환해야 합니다. Date 오브젝트 개요 - 년월일, 시분초, 밀리초 제공 - 시간 값 (Time value) 라고 부름 - UTC (Universal Time Coordinated) 기준 - 1970년 1월 1일 기준으로 밀리초를 제공 남는 초는 무시 - 1970.01.01 0시 기준으로 전후 100,000,000일 지원 - UTC와 GMT(Greenwich Mean Time) - JS는 UTC 기준 - 시간 값 표시 기준 - 월은 0부터 시작 - 일은 1에서 31일가지 정수로 표시 - 요일은 0부터 시작 (0: 일요일, 1: 월요일, 6: 토요일) Date 오브젝트 프로퍼티 - Date() - 현재 시각 반환 - Date() - Date() - Date() - Date()
