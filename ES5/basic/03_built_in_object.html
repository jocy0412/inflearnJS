빌트인 - Built-in이란 값 타입, 연산자, 오브젝트(object)를 사전에 만들어 놓은 것 JS 코드를 처리하는 영역에 - 장점 사전 처리를 하지 않고 즉시 사용 자바스크립트 특징 - 빌트인 값 타입 Undefined, Null, Boolean, Number, String, Object - 빌트인 연산자 +, -, *, /, %, ++, --, new 등 - 빌트인 오브젝트 (10 여개의 빌트인 오브젝트가 ES5에 있음) - 빌트인 Number 오브젝트 Number 처리를 위한 프로퍼티의 집합, 오브젝트 입니다. 123과 같은 숫자, 상수, 지수를 처리하는 오브젝트 여기서 오브젝트는 소문자 object 대문자 Object는 값을 저장하는데 중점이고 소문자 object는 값을 처리하는 것이 중점이다. - 빌트인 오브젝트 유형 Number 오브젝트 123과 같은 숫자, 상수, 지수 String 오브젝트 abc와 같은 문자열, 분리, 연결 Boolean 오브젝트 true, false Object 오브젝트 {key: value} Array 오브젝트 [1, 2, "A", "가나다"] 형태 Function 오브젝트 function abc(){} 형태 Math 오브젝트 abs(), round() 등의 수학 계산 Date 오브젝트 연월일, 시분초 JSON 오브젝트 [{"name":"value"}] 형태 서버와 데이터 송수신에 사용 RegExp 오브젝트 ^, $와 같은 정규 표현식 본 강좌에서 다루지 않음 Global 오브젝트 소스 파일 전체에서 하나만 존재 모든 코드에서 공유, 접근 가능 전역 객체라고도 하며, 뉘앙스에 차이 있음
